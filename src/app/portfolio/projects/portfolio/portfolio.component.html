Portfolio (Angular)

• Description : The Portfolio is a dynamic and responsive Single Page Application (SPA) built using Angular. It serves as a personal portfolio to showcase your professional background, skills, projects, and contact information. The application is organized into multiple components, each focusing on a specific section of the portfolio, with seamless navigation using Angular Routing.

• Key Components:

> Dashboard: The landing section of the app, giving a quick overview of the portfolio.

> Profile: Displays personal details, bio, and other introductory information.

> Education: Highlights academic qualifications and educational background.

> Experience: Showcases professional work experience, roles, and responsibilities.

> Projects: A main component that contains sub-components, each representing individual projects. Each project includes:
      -- A brief description
      -- A GitHub link for source code or live preview

> Contact: A contact form or section where users can get in touch.

> SideNav: A side navigation panel that enables smooth, user-friendly navigation across all sections of the app.

Routing: Angular’s routing module is implemented to allow users to navigate between different sections without reloading the page. The SideNav makes it easy to access each part of the portfolio instantly, enhancing the overall user experience.
