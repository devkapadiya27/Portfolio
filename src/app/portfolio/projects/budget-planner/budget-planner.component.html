Budget Planner (Angular)

• Description : The Budget Planner is a Single Page Application (SPA) built using Angular. It is designed to help users manage their finances effectively by tracking income, expenses, tasks, and more. The application is composed of multiple reusable and modular components, each handling a specific functionality.

• Key Components:

Dashboard: Provides a summary view of the user's financial status, including total income, expenses, and balance.

Income: Allows users to add, update, and view their income sources.

Expense: Enables users to log and categorize expenses for better financial tracking.

History: Displays a transaction history, including past income and expenses, with filtering and sorting options.

ToDo: Lets users create and manage daily financial tasks or reminders.

Login: Handles user authentication for secure access to the application.

Profile: Allows users to manage their personal information and settings.

SideNav: A side navigation menu that helps users navigate between different components easily.

Routing: Angular's routing module is used to navigate between components without reloading the page, providing a smooth and dynamic user experience.Each component is mapped to a specific route, and the SideNav makes switching betweenthese views intuitive and seamless.
